F.A.Q.

What does Bitdefender Safety Check do?

Bitdefender Safety Check is a free vulnerability scanner. It continuously scans your Windows PC for proof of compromise and vulnerable user settings to let you know when you are at risk.

What type of devices and operating systems does the tool support?

Bitdefender Safety Check is only available to scan Windows PCs.

I already have a security solution installed. Will my system be impacted?

Bitdefender Safety Check will not cause any additional slow-down to your system.

Do I have to uninstall my current security solution?

There is no need to immediately uninstall your current security solution. Bitdefender Safety Check acts as a second opinion scanner, working seamlessly alongside any other security solution.

What is a “second opinion” scanner?

Bitdefender Safety Check makes sure you get notified if vulnerabilities are found on your system and your primary security solution fails to protect you from all threats.

What types of threats are detected?

Bitdefender Safety Check detects if anything has slipped by your current security solution and left a mark on your system. It also detects if something is wrong with your current configuration so you can close security gaps.

How does it prevent additional malware from infecting my system?

Although it does not protect you against malware on its own, Bitdefender Safety Check lets you test Bitdefender’s world-class antimalware solutions.

You can try Bitdefender Total Security for 30 days (totally free) and see the best detection technologies in action.

Is Bitdefender Safety Check really free?

Bitdefender Safety Check is completely free. There is no expiration date tied to it, and you can install this free vulnerability scanner on as many Windows PCs as you want.

Do I need a Bitdefender account to use Bitdefender Safety Check?

No, there is no need for you to sign up. Just install the free tool and start using it.

Can I use it if I already have a Bitdefender solution installed on my device?

If you have Bitdefender on your system, it means that you already enjoy the highest level of protection we can offer. All the components and technologies of Bitdefender Safety Check are already included in your current product.
