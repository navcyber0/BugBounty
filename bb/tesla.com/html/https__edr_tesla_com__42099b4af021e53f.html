Tesla Event Data Recorder (EDR) Resources

To access EDR data from a Tesla vehicle you’ll need a Windows computer, appropriate cables and adapters, the Tesla EDR retrieval program and a MyTesla account. See the help page for more information. Please use the Tesla EDR website corresponding to your region (North America, EU, China).
